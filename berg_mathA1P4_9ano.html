FUNÇÃO do 1º GRAU (y = ax + b)

  • a = coeficiente angular: a > 0 reta é crescente, a < 0 reta é decrescente, a = 0 a reta é linear.
  • b = coeficiente linear: é o ponto da reta que intercede com o eixo y e tem coordenadas (0, b)
  • Num gráfico onde há 2 retas que se intercedem, o X e o Y daquele ponto de interseção são descobertos igualando as funções das 2 retas

FUNÇÃO do 2º GRAU (y = ax² + bx + c)

Δ = b² - 4ac

  • Δ = 0 -> a função tem 1 Raiz Real
  • Δ > 0 -> a função tem 2 Raízes Reais
  • Δ < 0 -> a função tem 0 Raízes Reais

As raízes são as intersecções da parábola com o eixo X

Se a > 0 a função tem concavidade pra cima (U), se a < 0 ela tem concavidade pra baixo (n)

X1.X2 = c/a e X1 + X2 = -b/a, onde X1 e X2 são as raízes

O vértice da parábola é o ponto mais alto ou mais baixo e tem coordenadas (-b/2a; -Delta/4a)

O coeficiente c é o ponto da parábola que intercede com o eixo y de coordenadas (0, c)

Existe uma forma fatorada da função de 2º grau, onde y = a.(x - x1).(x - x2)

LEIS do SENO e do COSSENO

  • Cosseno: a² = b² + c² - 2.b.c.(cos a) -> que é o cosseno do ângulo oposto a 'a'
  • Seno: a/Sen A = b/Sen B = c/Sen C -> cada um desses senos gemado do ângulo oposto ao seu respectivo lado

-> OBS1: Cos (Complemento de alpha) = - Cos alpha

-> OBS2: Sen (Suplemento de alpha) = Sen alpha

CIRCUNFERÊNCIA

2.pi.r / um arco = 360° / o arco em graus

  • O comprimento da circunferência é 2.pi.Raio, em unidade de comprimento, mas em radiano é só 2.pi.
  • Uma relação métrica entre cordas de uma circunferência que se cruzam é a seguinte, se AB e CD forem cordas e O é o ponto onde elas se cruzam:

AS/CS = DS/BS

AS.BS = CS.DS

-> OBS1: vale lembrar que o menor lado de um triângulo é sempre oposto ao menor ângulo, e o mesmo serve pro maior lado com maior ângulo.
